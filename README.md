# Homebrew Tap for Caged

Official Homebrew formulae for [Caged](https://caged.dev) tools.

## Installation

```bash
brew tap caged-dev/tap
brew install caged-agent
brew install caged-mcp-server
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `caged-agent` | Lightweight agent that runs inside Caged sandbox VMs |
| `caged-mcp-server` | Model Context Protocol server for AI coding agents |

## Updating

```bash
brew update
brew upgrade caged-agent
brew upgrade caged-mcp-server
```
